# WezTerm キーバインド一覧

> **リーダーキー**: `Ctrl+T`（2秒以内に次のキーを押す）

---

## ウィンドウ操作

| キー | 動作 |
|------|------|
| `Alt+Enter` | フルスクリーン切り替え |
| `Cmd+N` | 新しいウィンドウを開く |

---

## タブ操作

| キー | 動作 |
|------|------|
| `Cmd+T` | 新しいタブを開く |
| `Cmd+W` | 現在のタブを閉じる（確認あり） |
| `Ctrl+Tab` | 次のタブへ |
| `Ctrl+Shift+Tab` | 前のタブへ |
| `Cmd+1〜9` | タブ番号で切り替え（9は最後のタブ） |

---

## ペイン操作

### 分割・クローズ

| キー | 動作 |
|------|------|
| `Ctrl+T` → `R` | 横分割（左右） |
| `Ctrl+T` → `D` | 縦分割（上下） |
| `Ctrl+T` → `X` | 現在のペインを閉じる |

### フォーカス移動

| キー | 動作 |
|------|------|
| `Ctrl+T` → `H` | 左のペインへ |
| `Ctrl+T` → `L` | 右のペインへ |
| `Ctrl+T` → `K` | 上のペインへ |
| `Ctrl+T` → `J` | 下のペインへ |

### ズーム

| キー | 動作 |
|------|------|
| `Ctrl+Z` | アクティブペインをズーム切り替え（ペインが2つ以上の時のみ） |

### サイズ調整（Setting モード）

`Ctrl+T` → `S` でSettingモードに入ってから:

| キー | 動作 |
|------|------|
| `H / L / K / J` | ペインサイズを1単位調整（左/右/上/下） |
| `1〜9` | ペインの**高さ**を10%〜90%に設定 |
| `Ctrl+1〜9` | ペインの**幅**を10%〜90%に設定 |
| `Escape` / `T` / `Ctrl+C` | Settingモードを抜ける |

---

## コピー・ペースト

| キー | 動作 |
|------|------|
| `Cmd+C` | コピー |
| `Cmd+V` | ペースト |
| `Ctrl+C` | コピー（ターミナルスタイル） |
| `Cmd+Space` | クイックセレクト（画面上のURL・パスなどを選択） |
| `Ctrl+T` → `Z` | 直前のコマンドと出力をコピー |

### コピーモード（`Ctrl+X` で起動）

Vimライクなキーで画面テキストを選択・コピーできます。

| キー | 動作 |
|------|------|
| `H/J/K/L` | カーソル移動（←↓↑→） |
| `W / B / E` | 単語単位で移動 |
| `0 / ^ / $` | 行頭・非空白行頭・行末 |
| `G / GG` | 最下部 / 最上部 |
| `V` | 行選択モード |
| `Ctrl+V` | ブロック選択モード |
| `V`（小文字） | 文字選択モード |
| `Y` | ヤンク（コピー） |
| `/` | 検索 |
| `T` / `Escape` / `Ctrl+C` | コピーモード終了 |

---

## スクロール

| キー | 動作 |
|------|------|
| `Shift+PageUp/Down` | 1ページスクロール |
| `Alt+P` / `Alt+N` | 0.5ページスクロール（上/下） |
| `Cmd+↑` | 0.5ページ上にスクロール |
| `Cmd+↓` | 0.5ページ下にスクロール |
| `Alt+[` / `Alt+]` | 前のプロンプト / 次のプロンプトへジャンプ |

---

## 検索

| キー | 動作 |
|------|------|
| `Cmd+F` | 検索を開く |
| `Ctrl+N` / `Ctrl+P` | 次の一致 / 前の一致（検索中） |
| `Ctrl+R` | 検索タイプを切り替え（テキスト/正規表現） |
| `Escape` | 検索を閉じる |

---

## フォントサイズ

| キー | 動作 |
|------|------|
| `Cmd++` | フォントサイズを大きく |
| `Cmd+-` | フォントサイズを小さく |
| `Cmd+0` | フォントサイズをリセット |

---

## コマンドランチャー

| キー | 動作 |
|------|------|
| `Ctrl+T` → `O` | ランチャー（lazygit / nvim / yazi など） |
| `Ctrl+T` → `B` | バッファの内容をNeovimで表示 |
| `Ctrl+P` | コマンドパレット |

---

## その他

| キー | 動作 |
|------|------|
| `Ctrl+T` → `A` | AWS ARNをTuickSelectしてAWSコンソールで開く |
| `Cmd+R` / `Ctrl+R` | 設定をリロード |
| `Cmd+L` | デバッグオーバーレイを表示 |
| `Ctrl+U` | Unicode文字入力パレット |
| `Shift+Enter` | 改行を送信（Claude Codeで複数行入力） |
| `Alt+¥` | バックスラッシュ `\` を送信 |
