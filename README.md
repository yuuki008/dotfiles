## シンボリックリンク

- nvim

```
ln -s ~/github/yuuki008/dotfiles/.config/nvim ~/.config/
```

- zshrc

```
ln -s ~/dotfiles/.zshrc ~/.zshrc
```

- note

```
ln -s ~/github/yuuki008/dotfiles/scripts/note /usr/local/bin/note
```
